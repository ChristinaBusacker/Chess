A little Chess-Project

About

Unfortunately I only had a very limited time for this project. I chose chess because a lot of things here are not dependent on interpretations. That would be different with a pure fantasy project. The game is of course not finished. Castling is not possible, nor are moves of the king unblocked if this would put him in check.

Used Frameworks
